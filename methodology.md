# Methodology

## Problem Formulation
We formulate the central task for **Secrecy Capacity Analysis and Countermeasures for Indoor VLC** through the governing equation below.

$$C_s = \max\!\left(0,\; B\log_2\!\left(1+\mathrm{SNR}_B\right)-B\log_2\!\left(1+\mathrm{SNR}_E\right)\right)$$

## Variable Definitions
| Symbol | Definition | Value |
|---|---|---|
| C_s | Secrecy capacity | Derived |
| B | Optical channel bandwidth | 20 MHz |
| \mathrm{SNR}_B | Legitimate receiver SNR | Derived |
| \mathrm{SNR}_E | Eavesdropper SNR | Derived |

## Evaluation Logic
The repository is framed around benchmark-aware comparison, reproducible parameterization, and professor-friendly reporting.
